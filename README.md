# Installation
`npm i split-panel`

# Usage

```javascript
import SplitPane from 'SplitPane';
````
```javascript
<SplitPane>
  <div>a</div>
  <div>b</div>
</SplitPane>
```

## Props
#### leftMaxWidth ([number] default: `null`)

Left panel maximum width in pixel
    
####  leftMinWidth ([number], default: `null`)

Left panel minimum width in pixel
